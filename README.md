# Nova Estate

Nowoczesny landing page dla fikcyjnej firmy deweloperskiej, zbudowany jako projekt portfolio prezentujący pracę z Next.js App Router i animacjami sterowanymi scrollem.

## Technologie

- Next.js 16 (App Router, React Server Components)
- React 19
- TypeScript
- Tailwind CSS v4
- Motion (Framer Motion) - animacje scrollowe, parallax, przejścia

## Funkcje

- Animowany hero z efektem paralaksy powiązanym ze scrollem
- Licznik statystyk animowany przy wejściu w widok
- Filtrowana siatka inwestycji (w budowie / gotowe / planowane)
- Animowana oś czasu procesu zakupu mieszkania
- Karuzela opinii klientów
- Formularz kontaktowy z walidacją i stanem wysłania
- W pełni responsywny układ, sticky nawigacja z mobilnym menu

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

Aplikacja wystartuje na [http://localhost:3000](http://localhost:3000).

## Status

Projekt ukończony - wersja demonstracyjna do portfolio.
